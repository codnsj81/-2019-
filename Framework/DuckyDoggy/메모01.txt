게임 프로그램이 가져야 하는 기본적인 구성 요소
- 윈도우 메시지 처리
	윈도우 메뉴와 프로그램 종료 그리고 윈도우 메시지 처리
- direct3D 디바이스 생성
	direct3D 그래픽 디바이스, 스왑 체인, 렌더 타겟 등을 생성하여 그래픽 출력 준비
- direct3D 디바이스 소멸
	프로그램이 종료될 때 생성하였던 Direct3D 그래픽 디바이스, 스왑체인, 렌더타겟 등을 소멸 시킴
- 사용자 입력 처리
	사용자 입력(미우스 , 키보드)을 처리하여 게임 캐릭터의 이동 또는 게임 객체의 움직이믕ㄹ 제어하거나
	게임에 필요한 입력 처리, DirectInput을 사용하거나 윈도우 API 함수 사용
- 게임 객체 생성
	게임에서 사용되는 게임 객체를 생성
- 게임 객체의 소멸
	게임에서 더 이상 사용되지 않는 게임 객체를 소멸
- 게임 객체의 에니메이션 처리
	사용자 입력을 기반으로 게임 객체의 움직임 또는 에니메이션 처리
- 게임 상태 정보 설정
	게임의 진행과 그래픽 렌더링의 관리를 위한 상태 정보 설정
- 장면 생성 및 렌더링
	게임의 한 장면을 생성하여 그래픽 디바이스에 출력
